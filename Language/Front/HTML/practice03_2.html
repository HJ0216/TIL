다양한 input type
input type="text": 한 줄 텍스트를 입력할 수 있는 요소 생성


input type="password": 비밀번호를 입력할 수 있는 요소 생성


input type="tel": 전화번호 형식을 입력할 수 있는 요소 생성


input type="number": 숫자만 입력할 수 있는 요소 생성


input type="url": url 주소 형식을 입력할 수 있는 요소 생성


input type="search": 검색용 텍스트를 입력할 수 있는 요소 생성


input type="email": 이메일 형식을 입력할 수 있는 요소 생성


input type="checkbox": 체크박스 요소 생성


input type="radio": 라디오버튼 요소 생성


input type="file": 파일 업로드 요소 생성


input type="button": 버튼 요소 생성


input type="image": 이미지로 버튼 요소 생성(src 속성 사용 O, alt는 사용 X)


input type="hidden": 사용자 눈에 보이지 않는 입력 요소 생성


input type="date": 연, 월, 일을 선택할 수 있는 입력요소를 생성


input type="datetime-local": 연, 월, 일, 시, 분을 선택할 수 있는 요소 생성


input type="month": 연, 월을 선택할 수 있는 요소 생성


input type="week": 연, 주차를 선택할 수 있는 요소 생성


input type="time": 시간을 선택할 수 있는 요소 생성


input type="range": 숫자 범위를 선택할 수 있는 슬라이드 요소를 생성


input type="color": 색상을 선택할 수 있는 요소 생성


input type="summit": 폼 전송 역할을 하는 버튼 요소 생성


input type="reset": 폼 요소에 사용자가 입력한 값을 초기화하는 버튼 요소 생성
